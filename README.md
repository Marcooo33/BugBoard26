### Description
**BugBoard26** is an issue-handling platform designed for software projects. The system facilitates efficient development team coordination and streamlined bug reporting.

### Key Features
* **Issue Management:** Report and track various issue types (Bug, Feature, Documentation,  Question).
* **Interactive Comments:** Actively engage with issues by **commenting** and **modifying** core attributes such as **status** and **priority**.
* **User Management:** A structured user hierarchy featuring **Viewer**, **Developer**, and **Admin** roles.

### Demo Credentials
To explore the platform's features, use the following login credentials:
* **Email:** `developer@email.com`
* **Password:** `developer`

### Tech Stack
The application is built on a **microservices-based Client-Server architecture**:
* **Frontend:** Angular (TypeScript).
* **Backend:** Java Spring (decoupled into User and Issue Management services).
* **Database:** PostgreSQL.
